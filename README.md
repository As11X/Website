# Читай Город — интернет-магазин книг

Итоговый проект по дисциплине «Проектирование, разработка и оптимизация веб-приложений».

## Хостинг (GitHub Pages)

1. Создайте репозиторий на GitHub и загрузите проект
2. Settings → Pages → Source: `main` branch, folder `/ (root)`
3. Сайт будет доступен по адресу `https://<username>.github.io/<repo>/`

## Технологии

HTML5, CSS3, JavaScript (ES6 modules), JSON, localStorage
